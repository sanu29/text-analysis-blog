NPM packages for Text Analysis

Introduction

Text analysis can be said as extraction of important keywords and then classifying them into categories which let us know more about the text. Such as extracting the keywords. Get if the text or paragraph is negative or positive. The count of characters, words. To get the verbs, nouns and adjective. There are many algorithms used in this but with some npm packages make it easy for us to implement text analysis within few lines of code. Here we will cover some basic text analysis using NPM packages.

  • Keyword Extraction:

    This is a process in which we extract the important keywords from the text or paragraph.
    Package name: keyword-extractor
    Installation: npm install keyword-extractor
    Function : keyword-extractor.extract(input)
    This package will take text as input and will return an array of keywords.
    For more details you can visit: https://www.npmjs.com/package/keyword-extractor

  • Sentimental Analysis:

    In sentimental analysis we determine if the provided paragraph is positive or negative or neutral.
    Package name: sentiment
    Installation : npm install sentiment
    Function : sentiment.analyze(text);
    This package will take input block and return the words which are positive or negative and also will return comparative percentage. For more visit: https://www.npmjs.com/package/sentiment

  • Parts of speech:

    In parts of speech, we determine the nouns, adjectives and verbs in the sentence.
    Package name: wordpos
    Installation : npm install -g wordpos
    Function : wordpos.getPOS(input, callback);
    This will take input as text block and return an array of parts of speech.

Conclusion

Using these npm packages you can easily proceed with text analysis rather than writing and implementing the algorithms and logic again from start.
